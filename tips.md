## 1. Математика

### 1.1. Квадратный трёхчлен: сумма и произведение корней

**Задача:**  
Дан трёхчлен `ax² + bx + c`. Найти сумму или произведение корней.  
**Решение:**  
- Сумма корней: `-b/a`  
- Произведение корней: `c/a`  

**Где используется:**  
2024, задача №1–3: `x² – 12x + 7` → сумма корней `12`, произведение `7`, и т.д.

### 1.2. Последняя цифра суммы квадратов

**Задача:**  
Найти последнюю цифру суммы `1² + 2² + ... + n²`.  
**Решение:**  
Сумма квадратов по модулю 10 повторяется с периодом 10. Можно вычислить остаток от деления n на 10 и использовать заранее вычисленный паттерн.  
**Код:** `math/number_theory.py` – `sum_of_squares_last_digit(n)`.  

**Пример:** 2025, №2.

### 1.3. НОД чисел вида 10^a – 1 и 10^b – 1

**Задача:**  
Найти `НОД(10^a – 1, 10^b – 1)`.  
**Решение:**  
`НОД(10^a – 1, 10^b – 1) = 10^{НОД(a,b)} – 1`.  
**Для суммы:** `НОД(10^a + 1, 10^b + 1) = 10^{НОД(a,b)} + 1` (если a и b нечётны, иначе 1).  
**Код:** `math/number_theory.py` – `gcd_power_minus_one`, `gcd_power_plus_one`.  

**Пример:** 2023, №9–10.

### 1.4. Сумма всех пятизначных чисел

**Задача:**  
Найти сумму всех пятизначных натуральных чисел.  
**Решение:**  
Арифметическая прогрессия: от 10000 до 99999, количество `n = 90000`.  
Сумма = `(первый + последний) * n / 2`.  
**Код:** `math/combinatorics.py` – `sum_of_n_digit_numbers(digits)`.  

**Пример:** 2023, №7.

### 1.5. Телескопические суммы

**Задача:**  
Вычислить `2025 * (1/(20*21) + 1/(21*22) + ... + 1/(2024*2025))`.  
**Решение:**  
`1/(k(k+1)) = 1/k – 1/(k+1)`. Сумма телескопируется.  
**Код:** `math/series.py` – `telescoping_product_sum(start, end)`.  

**Пример:** 2025, №4.

## 2. Комбинаторика и вероятность

### 2.1. Перестановки слов с ограничениями

**Задача:**  
Сколько различных слов можно получить, переставляя буквы в слове «ВЫЗОВЫ», если слово не может начинаться с буквы «Ы»?  
**Решение:**  
Общее число перестановок с повторениями: `N = n! / (∏ k_i!)`.  
Число перестановок, начинающихся на запрещённую букву: фиксируем её в начале, считаем перестановки остальных.  
**Код:** `math/combinatorics.py` – `permutations_with_repetitions`, `words_not_starting_with`.  

**Пример:** 2025, №3.

### 2.2. Вероятность суммы на двух кубиках, делящейся на 4

**Задача:**  
При броске двух шестигранных кубиков найти вероятность, что сумма делится на 4.  
**Решение:**  
Всего исходов 36. Перебрать все пары (1..6, 1..6) и посчитать подходящие.  
**Код:** `probability/probability.py` – `dice_sum_probability(target_sum, dice=2, sides=6)`.  

**Пример:** 2025, №5.

### 2.3. Вероятность деления на 3, но не на 12

**Задача:**  
Из чисел 1..500 случайно выбирается одно. Какова вероятность, что оно делится на 3 и не делится на 12?  
**Решение:**  
Чисел, делящихся на 3: `500 // 3`.  
Чисел, делящихся на НОК(3,12)=12: `500 // 12`.  
Искомая вероятность = `(500//3 – 500//12) / 500`.  
**Код:** `probability/probability.py` – `probability_divisible_by_n`.  

**Пример:** 2025, №6.

### 2.4. Браслет из бусин (максимальная длина)

**Задача:**  
Даны бусины разных цветов. Нужно составить браслет, где соседние бусины разного цвета. Какое максимальное количество бусин можно использовать?  
**Решение:**  
Если самый частый цвет > (сумма всех остальных + 1), то можно использовать только `2*сумма_остальных + 1`.  
Иначе – все бусины.  
**Код:** `math/combinatorics.py` – `max_bracelet_length(counts)`.  

**Пример:** 2025, №8.

### 2.5. Строки длины 10 из 0 и 1 без двух нулей подряд

**Задача:**  
Сколько существует строк длины 10 из цифр 0 и 1, не содержащих двух нулей подряд?  
**Решение:**  
Это числа Фибоначчи: `F(n+2)`. Для n=10: `F(12)`.  
**Код:** `math/combinatorics.py` – `binary_strings_without_adjacent_zeros(n)`.  

**Пример:** 2025, №9.

### 2.6. Распределение яблок

**Задача:**  
Сколькими способами можно распределить 10 одинаковых яблок между тремя людьми (возможно, кому-то ничего не достанется)?  
**Решение:**  
Звёзды и палочки: `C(10+3-1, 3-1) = C(12,2)`.  
**Код:** `math/combinatorics.py` – `stars_and_bars`.  

**Пример:** 2023, №10.

### 2.7. Выбор команды (мальчики и девочки)

**Задача:**  
В классе 8 мальчиков и 9 девочек. Сколькими способами выбрать 3 мальчиков и 2 девочек?  
**Решение:**  
`C(8,3) * C(9,2)`.  
**Код:** `math/combinatorics.py` – `comb`.  

**Пример:** 2023, №9; 2024, №7–9.

## 3. Графы

### 3.1. Чтение матрицы смежности

**Задача:**  
Дан файл `graph.txt`, где каждая строка – строка матрицы смежности (0/1 через запятую). Найти вершину с наибольшим числом рёбер.  
**Решение:**  
Прочитать, построить список смежности, посчитать степени.  
**Код:** `algorithms/graph_utils.py` – `read_adj_matrix`, `matrix_to_adj`, `vertex_max_degree`.  

**Пример:** 2024, №14–16.

### 3.2. Компоненты связности

**Задача:**  
Найти число компонент связности неориентированного графа.  
**Решение:**  
DFS или BFS по всем вершинам.  
**Код:** `algorithms/graph_utils.py` – `connected_components`.  

**Пример:** 2024, №17.

### 3.3. Кратчайший путь

**Задача:**  
Для заданных пар вершин найти длину кратчайшего пути (или факт существования пути).  
**Решение:**  
BFS из каждой начальной вершины (невзвешенный граф).  
**Код:** `algorithms/shortest_path.py` – `bfs_shortest_path`.  

**Пример:** 2023, №18; 2024, пары вершин.

## 4. Анализ данных (pandas)

### 4.1. Загрузка и объединение таблиц

**Задача:**  
Даны несколько CSV (user, activity, message, video, post). Нужно объединить их для расчёта индекса активности.  
**Решение:**  
- `pd.read_csv` для каждого файла.  
- Сначала объединить activity с message по `activity_id`, аналогично с video и post.  
- Затем объединить с user по `user_id`.  

**Код:** `data_analysis/load_data.py` – `merge_tables`.

### 4.2. Индекс активности

**Задача:**  
Рассчитать баллы для каждого действия (сообщения, видео, посты) и усреднить по активным пользователям.  
**Решение:**  
Применить правила начисления баллов, сгруппировать по user_id, вычислить среднее.  
**Код:** `data_analysis/activity_index.py`.

### 4.3. Возраст пользователей

**Задача:**  
Вычислить возраст на 1 января 2025 года по дате рождения (формат YYYY-mm-dd).  
**Решение:**  
`(pd.Timestamp('2025-01-01') - pd.to_datetime(birth_date)).dt.days // 365`.  
**Код:** `data_analysis/load_data.py` – `age_on_date`.

### 4.4. Заполнение пропусков средним по группе

**Задача:**  
У пользователей с пропущенной датой рождения восстановить возраст как средний возраст в группе по индексу активности.  
**Решение:**  
Сгруппировать пользователей с известным возрастом по группам активности, вычислить средний возраст в группе, применить к пропускам.  
**Код:** `data_analysis/impute_age.py` (можно создать).

## 5. Машинное обучение

### 5.1. Линейная регрессия и R²

**Задача:**  
Дана модель `ŷ = 0.2 + 0.04·x`. Построить график, вычислить R² на данных.  
**Решение:**  
- R² = 1 – (сумма квадратов ошибок) / (сумма квадратов относительно среднего).  
- Для подбора оптимальных параметров можно искать `w0` и `w1`, максимизирующие R² (перебор или МНК).  
**Код:** `machine_learning/linear_regression.py` – `r2_score`, `fit_linear_regression`.

**Пример:** 2025, №1 (в блоке ML), задачи на R².

### 5.2. Логистическая регрессия и подбор порога

**Задача:**  
Дана модель с параметрами `w0, w1, w2` и порогом `p`. Найти accuracy на валидационной выборке. Подобрать `p` для максимизации accuracy.  
**Решение:**  
- Вычислить `σ(w0 + w1·x1 + w2·x2)`.  
- Предсказать класс по сравнению с `p`.  
- Для подбора порога перебирать `p` от 0 до 1 с шагом 0.01.  
**Код:** `machine_learning/logistic_regression.py` – `logistic_predict`, `find_best_threshold`.

**Пример:** 2024, №22–24.

### 5.3. Кусочно-константная модель

**Задача:**  
Разбить x на интервалы [0,1), [1,2), … [9,10). В каждом интервале предсказывать среднее y. Найти R².  
**Решение:**  
Использовать `pd.cut` для разбиения, затем `groupby` и `map`.  
**Код:** `machine_learning/piecewise_models.py` – `piecewise_constant`.

**Пример:** 2025, №4 (в блоке ML).

### 5.4. Кусочно-линейная модель

**Задача:**  
На каждом интервале построить свою линейную регрессию.  
**Решение:**  
Для каждого интервала отфильтровать данные, обучить `LinearRegression`. При предсказании применить нужную модель.  
**Код:** `machine_learning/piecewise_models.py` – `piecewise_linear`.

**Пример:** 2025, №5 (в блоке ML).

## 6. Программирование (Python)

### 6.1. Поиск ошибок в коде

**Задача:**  
Дан фрагмент Python-кода с ошибками. Найти номера строк.  
**Типичные ошибки:**  
- Неправильный режим открытия файла (`'w'` вместо `'r'`).  
- Неверный доступ к элементам списка (`arr[j][i]` вместо `arr[i][j]`).  
- Пропущенные отступы (блок `if` без тела).  
- Использование `break` в неправильном месте.  
- Переменная не определена.  

**Где встречается:** 2024, №12–13.

### 6.2. Работа с zip-архивами и файлами

**Задача:**  
Внутри zip-архива лежат текстовые файлы. Нужно подсчитать общее количество букв или найти самые частые слова.  
**Решение:**  
- `zipfile.ZipFile` для чтения.  
- Для каждого файла прочитать, очистить текст (`.isalpha()`, `re.sub`), собрать статистику.  
**Код:** `nlp/text_processing.py` – `count_letters_in_zip`, `most_frequent_words_in_zip`.

**Пример:** 2024, №18–19.

### 6.3. Обработка CSV-файлов с действиями пользователей

**Задача:**  
Дан CSV с колонками `ip`, `date`, `action`. Найти дату с наибольшим числом пользователей, количество уникальных пользователей в марте 2023, число кликов.  
**Решение:**  
- Прочитать CSV, распарсить дату.  
- Сгруппировать по дате, посчитать уникальные ip.  
- Отфильтровать по марту 2023, взять уникальные ip.  
- Для кликов: разбить строку `action` по запятой, посчитать вхождения `'click'`.  
**Код:** `data_analysis/user_actions.py`.

**Пример:** 2024, №19–21.

## 7. Рекомендации по порядку действий на туре

1. **Оценить тип задачи.**  
   - Чистая математика → `math/`  
   - Комбинаторика/вероятность → `probability/`, `math/combinatorics.py`  
   - Графы → `algorithms/`  
   - Анализ данных (CSV) → `data_analysis/`  
   - ML → `machine_learning/`  

2. **Если даны данные:**  
   - Загрузить все таблицы в pandas.  
   - Проверить пропуски, типы колонок.  
   - Применить необходимые преобразования (даты, возраст).  

3. **Для ML-задач:**  
   - Начать с простой модели (линейная регрессия, среднее по интервалам).  
   - Если нужно точнее, использовать RandomForest или CatBoost.  
   - Для подбора параметров перебирать диапазоны, указанные в условии.  

4. **Округление:**  
   - Внимательно читать, до какого знака и в какую сторону округлять.  
   - Использовать `round(x, 2)`, `math.floor(x)`, `math.ceil(x)`.  

5. **Если результат не совпадает:**  
   - Проверить индексацию (0 vs 1).  
   - Проверить, учитываются ли пропуски.  
   - Убедиться, что правильно интерпретирована формула (например, скобки).
